# Combinations — Rosmarino

Four approved color combinations. Use these for layout sections, never mix tokens arbitrarily.

## Hero (dark, featured sections)
```css
background: var(--charcoal);
color: var(--cream);
/* eyebrow label */ color: var(--cognac-light);
/* CTA button */ background: var(--cognac); color: var(--cream);
```

## Card (product/content surfaces)
```css
background: var(--cream);
border: 1px solid var(--cream-dark);
color: var(--charcoal);
/* secondary text */ color: var(--stone);
/* CTA button */ background: var(--burgundy); color: var(--cream);
```

## CTA (promotional / conversion blocks)
```css
background: var(--burgundy);
color: var(--cream);
/* CTA button */ background: var(--cream); color: var(--burgundy);
```

## Nature (origin / sustainability sections)
```css
background: var(--forest);
color: var(--cream);
/* CTA button */ background: var(--cognac-light); color: var(--cream);
```

## Do not mix
- No burgundy text on cognac background.
- No forest background with burgundy CTA.
- No charcoal text on forest or burgundy — always use cream on dark backgrounds.
